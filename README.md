# Doctor_Recommendation_System
Recommends Doctor in Riga based on the symptoms which predicts the disease and recommends doctors score

Execution Steps:
1. Place the .py file and all the source files in the same Github path
2. login to Streamlit
3. Configure the application by pointing to Github path(Repository and .py file)
4. Once application built, select the symptoms from drop down as input
5. The result(Disease prediction and Doctor recommendation) will be displayed.
 
Input Files:

Original_Dataset.csv
Doctor_Versus_Disease.csv
Doctor_Specialist.csv
specialist_count.csv
riga_doctors_sample.xlsx
Disease_Description.csv

.py File: 
Doc_Rec_Git.py
